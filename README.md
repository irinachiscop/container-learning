# container-learning
My personal collection of resources for those wanting to learn more about Docker, Kubernetes, container security etc.

# Beginner
- [NIST Application container security guide](https://www.nist.gov/publications/application-container-security-guide)
- [Paper: Container Security: Issues, Challenges, and the Road Ahead (slightly old, but very good](https://ieeexplore.ieee.org/document/8693491)
- [Paper: On the Security of Containers: Threat Modeling, Attack Analysis, and Mitigation Strategies (has a good overview of cyberattacks as well)](https://www.sciencedirect.com/science/article/abs/pii/S0167404823000500)
- [What is container orchestration](https://www.logicmonitor.com/blog/what-is-container-orchestration)
- [Short article on differences between VMs and containers](https://learn.microsoft.com/en-us/virtualization/windowscontainers/about/containers-vs-vm)
- [Step-by-step tutorial for building basic understanding a basic understanding of Docker](https://github.com/andrewgmcritchie/learning-containers)
- [Beginner's guide to Kubernetes](https://k21academy.com/docker-kubernetes/kubernetes-for-beginners/)
- [Official Docker 101 tutorial](https://www.docker.com/101-tutorial/)


# Intermediate
- [O'Reilly book on Container Security](https://learning.oreilly.com/library/view/container-security/9781492056690/)
- [A curated repo of (mostly) tooling but also some articles on container security](https://github.com/kai5263499/awesome-container-security)
- [MITRE Att&ck framework for containers](https://attack.mitre.org/matrices/enterprise/containers/)

# Advanced
- [A very practical guide for devs on securing images and containers](https://github.com/krol3/container-security-checklist)
- [Framework for exploiting Docker containers](https://dockerexploitationframework.github.io/)
- [Falco: a real-time detection and monitoring solution for containers (amongst others)](https://falco.org/about/)
